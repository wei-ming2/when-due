## 🎯 Key Decisions Explained

### Why Tauri?
- 90% smaller than Electron (~12 MB vs ~150 MB)
- Native OS feel and performance
- Uses system WebView (no embedded Chromium)
- Perfect for lightweight productivity apps

### Why Svelte?
- Minimal framework (less boilerplate)
- Highly reactive (great DX)
- Smallest bundle size (~3 KB)
- Easy to learn while mastering fundamentals

### Why SQLite?
- Lightweight and reliable (used by major browsers)
- ACID-compliant (no data loss)
- Fast even with 10,000+ tasks
- Scales better than JSON as you grow
- No external server needed

### Why Rust?
- Type-safe even in systems code
- Excellent async/await support
- Zero-cost abstractions
- Memory-safe by default

### Why Custom CSS?
- Learning opportunity
- Full control over theme
- No framework bloat
- You understand how styling works
